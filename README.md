# Laravel Gallery Exam

## How to install
1. run
```
composer install
php artisan migrate
php artisan db:seed
```
2. run this if not yet installed
```
$ npm install -g @vue/cli
$ npm install -g @quasar/cli
```
3. build the quasar app
```
$ cd quasarapp
$ quasar build
```
4. run laravel mix on root folder
```
npm run dev
```
5. run serve
```
php artisan serve
```
